## 技術成果

### 後端架構 ✅
- **框架：** NestJS 11.x + TypeScript 5.7.x
- **資料庫：** TypeORM + SQLite/MySQL/PostgreSQL 支援
- **認證：** Passport.js JWT
- **事件系統：** NestJS EventEmitter
- **測試：** Jest + Supertest（1200+ 測試）

- **框架：** Vue 3.5 + Composition API
- **構建工具：** Vite 7.3.x
- **UI 組件庫：** Naive UI 2.43.x
- **狀態管理：** Pinia 3.0.x

- **多語言：** vue-i18n 9.14.x（zh-TW）
- **視覺化：** ECharts 6.0.x
- **測試：** Playwright（E2E）
