# Doctor CRM - 自費醫療管理系統

## 本項目概述

Doctor CRM 是一個為醫療機構設計的自費診療管理平台，專注於療程管理、醫令管理、費用追蹤，以及患者進度通知。系統支援多診所隔離、複雜的收入分配計算，以及未來與 Doctor Toolbox 的無縫整合。

## 核心價值

為醫療機構提供一個統一的平台，有效管理自費診療療程與醫令，同時提供清晰的財務透視和患者體驗提升，確保患者能即時接收治療進度提醒。

## 技術背景

### 現有技術棧

**後端：**
- NestJS 11.x + TypeScript 5.7.x
- TypeORM 0.3.x（支援 SQLite/MySQL/PostgreSQL）
- Passport.js JWT 認證
- NestJS Event Emitter（事件驅動）
- Jest + Supertest 測試

**前端：**
- Vue 3.5 + Composition API
- Vite 7.3.x 建構工具
- Naive UI 2.43.x（UI 組件庫）
- Pinia 3.0.x（狀態管理）
- Axios 1.13.x（API 客戶端）
- vue-i18n 9.14.x（傳統中文多語言）
- ECharts 6.0.x（資料視覺化）
- Playwright（E2E 測試）

**部署：**
- Docker + docker-compose
- GitHub Actions CI/CD

## 技術約束

- **資料庫**：支援 SQLite（開發）、MySQL/PostgreSQL（生產）
- **多租户**：內建多診所隔離邏輯
- **語言**：傳統中文（zh-TW）為主要語言
- **認證**：JWT 無狀態認證
- **事件系統**：使用 NestJS EventEmitter 解耦模組通訊
