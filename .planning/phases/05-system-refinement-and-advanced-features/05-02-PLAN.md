<objective>
Implement a Dynamic Attributes system (Custom Fields) to allow clinics to extend Patient and Treatment data models without code changes.

Purpose:
- Provide flexibility for different medical specialties to track specific data.
- Avoid "schema rot" by using JSONB for sparse, clinic-specific fields.

Output:
- Attribute Definition management (CRUD).
- JSONB storage integration for Patients and Treatments.
- Frontend dynamic form rendering.
</objective>
